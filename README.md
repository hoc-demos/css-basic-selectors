# CSS Basic Selectors

![](https://raw.githubusercontent.com/hoc-demos/images/main/css-basic-selectors.png)

### Type
```css
p {
    padding:20px;
}
```

### Class
```css
.p3 {
    background-color:purple;
}
```

### ID
```css
#p1 {
    background-color:orange;
}

#p2 {
    background-color: darkcyan;
}
```
